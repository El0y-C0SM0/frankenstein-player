\doxysubsection{Descrição detalhada}
Representa uma música com seus metadados. 

A classe \doxylink{classSong}{Song} armazena informações como título, artista, álbum, duração, gênero e ano de lançamento. Pode ser expandida para usar a biblioteca Tag\+Lib para extração automática de metadados de arquivos de áudio. Ademais, deve ser implementado a interface IPlayable
